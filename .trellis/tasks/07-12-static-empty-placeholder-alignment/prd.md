# Static Empty Placeholder Alignment

## Goal

Align pre-login / first-paint Keys, Logs, and Audit empty placeholders with the structured empty-state language used at runtime.

## Acceptance Criteria

- [x] Keys static tbody placeholder uses structured idle empty state.
- [x] Logs static tbody placeholder uses structured log empty state.
- [x] Audit static list placeholder uses structured audit empty state.
- [x] Static tests pin the new markup.
- [x] `npm run verify` passes.
