# Batch Bar Aria Labels

## Goal

Give batch-bar actions accessible names that state they apply to the **selected keys**, matching the clarity of `#batchClearSelection` and toolbar batch buttons.

## Requirements

- aria-label on enable/disable/reset/test selected actions.
- Keep short visible labels for density.
- Unit pins; verify green.

## Acceptance Criteria

- [x] Labels describe selected-key scope.
- [ ] Verify green.
