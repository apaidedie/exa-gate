# Export Buttons Accessible Labels

## Goal

Give `#exportLogs` and `#exportAudit` accessible names that state CSV export of the current filter window (not ambiguous “导出”).

## Requirements

- aria-label on both export toolbar buttons.
- Optional short visible text stay; logs may stay `导出` for density.
- Unit pins; verify green.

## Acceptance Criteria

- [x] aria-labels present.
- [ ] Verify green.
